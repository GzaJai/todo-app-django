# Todo App – Django

Aplicación de gestión de tareas desarrollada con **Django**, enfocada en practicar autenticación, CRUD y organización general del proyecto.

## Funcionalidades
- Registro y login de usuarios  
- Crear, editar y eliminar tareas  
- Marcar tareas como completadas  
- Interfaz sencilla basada en Django templates
